-Start file lines at the first instance of text
-permit or deny followed by the word 'any' or an ip address x.x.x.x 
	-currently only implemented for 192.168.1.x addresses
	-wildcards not implemented
where x = a number between 0 and 255
-Each line needs to have a separate rule
-deny by default
	-If there are only deny statements, then all other ip addresses will be permitted
	-If there is a mix of deny and permit, then only those addresses will be denied or permitted
